Full Stack Pizza

A Full Stack Javascript Project

A full stack application page based on a fictional pizzeria. Features on this page include: Being able to use the menu to filter options based on size and cost, add pizzas to a cart, modify the amount of pizzas of the same type in the cart, confirmation page with all the orders details

This project used a combination of React, CSS, HTML for the front-end, and Node.js, MongoDB for the back-end.

Skills Used: React, Node.js, Javascript, Mongoose, MongoDB, CSS, HTML
